# Política de Segurança

## Versões Suportadas

Atualmente, apenas a última versão da branch principal (`main`) recebe atualizações de segurança.

## Relatando uma Vulnerabilidade

Se você descobrir uma vulnerabilidade de segurança neste projeto, por favor, não abra uma Issue pública. Em vez disso, envie um e-mail para o mantenedor do projeto com detalhes sobre o problema para que possamos resolvê-lo de forma coordenada.
